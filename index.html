Sincronización entre dispositivos con GitHub Gist

GitHub Gist Sync
Sin configurar
¿Cómo sincronizar entre dispositivos?
1. Ve a github.com/settings/tokens
2. Crea un token con permiso gist
3. Pega el token abajo y haz clic en Guardar configuración
4. Haz clic en Guardar datos en nube — crea el Gist automáticamente
5. En tu otro dispositivo: pega el mismo token + el Gist ID y haz clic en Cargar desde nube
